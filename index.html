Backend Setup Instructions

This platform requires a Node.js + Express backend with PostgreSQL database.

1. Install Dependencies:

npm install express pg multer cors pdf-lib sharp uuid

2. Start the Server:

node server.js

3. Database Setup:

psql -U postgres -f schema.sql

The server runs on http://localhost:3000
